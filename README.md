# Catch The Fruit

Catch The Fruit is a browser-based arcade game built with HTML, CSS, and JavaScript. Catch falling fruits, avoid hazards, and aim for the highest score.

## Features

- Fast, arcade-style gameplay
- Sound effects and background music
- Score tracking and simple controls

## How to Run

1. Open `index.html` in a modern web browser.
2. Play and try to beat your high score.

## Project Structure

- `index.html` - Game layout and UI
- `style.css` - Styling and animations
- `script.js` - Game logic
- `sound/` - Audio assets
